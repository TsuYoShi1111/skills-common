# 分析ツール・思考プロセス・品質基準

## 分析の視点 — 必須確認事項

分析開始時に以下8項目を必ず確認する：

1. **課題タイプの分類**：戦略・オペレーション・技術・人事のどれか？
2. **課題の緊急度・重要度**：今すぐ解決すべきか？
3. **ステークホルダーの特定**：誰が影響を受け、誰が協力・反対するか？
4. **制約条件の整理**：予算・時間・技術・組織・法規制の制約
5. **成功の定義**：何をもって解決とするか？測定指標は？
6. **類似事例の確認**：過去の成功・失敗パターンはないか？
7. **外部環境の影響**：競合・市場・法規制・技術トレンドの影響
8. **組織の変革準備度**：変化を受け入れる準備ができているか？

## 思考プロセス

分析中に常に意識する6つのプロセス：

### 1. 課題の再定義
- 表面的な課題を鵜呑みにしない
- 「本当に解決すべき課題は何か？」を問い直す
- 症状と原因を混同していないか確認

### 2. 多角的検討
- 経営者・現場・顧客・競合の視点
- 短期と長期の両方の視点
- 定量と定性の両面から分析

### 3. システム思考
- 個別要素だけでなく全体の構造を見る
- 部分最適が全体最適を損なわないか確認
- 意図せぬ副作用・波及効果を予測

### 4. 実現可能性重視
- 理想論に走らず、実践可能な提案をする
- 組織の能力・文化に合った解決策を選ぶ
- 「今のリソースで実行できるか」を常に意識

### 5. 継続性考慮
- 一時的な対処でなく、持続可能な解決を目指す
- 仕組み化・制度化できる方法を優先
- 属人化を避ける設計

### 6. 学習・改善
- 実行結果を振り返る仕組みを組み込む
- 失敗からの学びを次に活かす
- ナレッジとして蓄積・共有する方法を提案

## 活用するフレームワーク一覧

### 課題分析系
| フレームワーク | 用途 | 使うタイミング |
|--------------|------|--------------|
| Why分析（5Why） | 根本原因の特定 | 課題の深掘り時 |
| ロジックツリー | 課題の構造化・分解 | 複雑な課題の整理時 |
| MECE分析 | 漏れなく重複なく整理 | 要素の分類・整理時 |
| フィッシュボーン図 | 原因の網羅的洗い出し | 多要因の課題分析時 |

### 戦略・環境分析系
| フレームワーク | 用途 | 使うタイミング |
|--------------|------|--------------|
| PEST分析 | 外部環境の把握 | 戦略課題の分析時 |
| 5Forces分析 | 業界の競争環境分析 | 市場参入・競争課題時 |
| 3C分析 | 市場・競合・自社の関係整理 | 戦略的意思決定時 |
| SWOT分析 | 強み弱み×機会脅威の整理 | 戦略オプション検討時 |

### 評価・意思決定系
| フレームワーク | 用途 | 使うタイミング |
|--------------|------|--------------|
| 優先順位マトリックス | 解決案の優先度付け | 複数案の比較時 |
| 影響度×関心度マトリックス | ステークホルダー分類 | 関係者分析時 |
| シナリオプランニング | 複数の将来像の検討 | 不確実性の高い課題時 |
| ROI分析 | 投資対効果の算出 | コスト判断が必要な時 |

## 出力品質チェックリスト

### 具体性・実践性
- [ ] 「改善する」ではなく「○○を○○により○○する」と記述しているか
- [ ] 可能な限り数値目標・期間を設定しているか
- [ ] 明日から実行できる行動項目を含んでいるか
- [ ] 見落としがちな要素も考慮しているか

### バランス・一貫性
- [ ] 理想と現実のバランスが取れているか
- [ ] 各セクション間で矛盾がないか
- [ ] 重要度に応じたメリハリがあるか
- [ ] 実行後の改善につながる設計になっているか

### アクション明確性
- [ ] 各アクションに「誰が」「いつまでに」「何をする」が明記されているか
- [ ] 優先順位が明確か（全部重要は意味がない）
- [ ] 必要リソースが具体的に記述されているか
- [ ] エスカレーション基準が定義されているか

## 課題規模に応じたセクション選択ガイド

### 小規模課題（影響範囲が限定的、1-2ヶ月で解決）
**必須**：セクション 1, 2, 5, 12
- 課題分析 → 解決案 → 優先順位 → ロードマップ

### 中規模課題（部門横断、3-6ヶ月）
**必須**：セクション 1, 2, 3, 4, 5, 6, 12
- 上記 ＋ クリティカル分析、ロジカル分析、シナリオ分析

### 大規模・戦略課題（全社的、6ヶ月以上）
**全セクション**：1〜12
- 全12セクションを網羅的に分析
