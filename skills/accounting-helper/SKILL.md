# 会計・帳簿管理サポート

事業の会計処理、帳簿整理、収支管理をわかりやすくサポートする。

## 重要な注意事項

- このスキルは会計処理の「ガイド」であり、税務アドバイスではない
- 最終的な判断は税理士・会計士に確認すること
- 税法の改正により内容が変わる場合がある

## ワークフロー

### Step 1: 状況確認

1. **事業形態**（個人事業主/法人）
2. **会計期間**（1月〜12月 / 4月〜3月）
3. **使用ツール**（freee/マネーフォワード/弥生/手書き）
4. **やりたいこと**（日常仕訳/月次締め/決算準備/確定申告）

### Step 2: 該当する処理を実施

## よく使う勘定科目

### 収入
| 科目 | 使う場面 | 例 |
|------|---------|-----|
| 売上高 | 本業の収入 | 制作費、コンサル料 |
| 雑収入 | 本業以外の収入 | キャッシュバック、ポイント |

### 経費（個人事業で多いもの）
| 科目 | 使う場面 | 例 |
|------|---------|-----|
| 通信費 | ネット、電話 | Wi-Fi、スマホ代 |
| 消耗品費 | 10万円未満の物品 | キーボード、文具 |
| 広告宣伝費 | 広告、PR | Google広告、チラシ |
| 外注費 | 外部への業務委託 | デザイン外注、ライティング |
| 旅費交通費 | 移動費 | 電車、タクシー |
| 接待交際費 | 取引先との飲食 | 会食、手土産 |
| 地代家賃 | 事務所家賃 | 按分計算が必要 |
| 水道光熱費 | 電気・ガス・水道 | 在宅の場合は按分 |
| 支払手数料 | 振込手数料、決済手数料 | |
| 減価償却費 | 10万円以上の資産 | PC、カメラ |
| 雑費 | 上記に分類できないもの | 多用しない |

## 確定申告チェックリスト

- [ ] 売上の計上漏れがないか
- [ ] 経費の領収書がすべて揃っているか
- [ ] 按分計算が適切か（家賃、通信費等）
- [ ] 減価償却の計算が正しいか
- [ ] 源泉徴収の記録があるか
- [ ] 社会保険料の控除資料があるか
- [ ] 小規模企業共済・iDeCo等の控除資料があるか
