# マーケティング施策レビュー

## レビュー対象

以下のマーケティング施策をレビューしてください: $ARGUMENTS

## 概要

LP構成・広告戦略・SEO戦略・集客計画を **5つの視点** で多角的にレビューし、現状スコア・改善案・改善後スコアを出力する。

## レビューフロー

```
対象文書の特定 → 施策タイプ判定 → 5視点レビュー → スコアリング → 改善案 → 改善後スコア → 統合レポート
```

## 対象施策タイプ

| タイプ | 例 |
|--------|-----|
| LP構成・設計 | セクション構成、ワイヤーフレーム、コピー |
| 広告戦略 | リスティング広告、SNS広告、ディスプレイ広告 |
| SEO戦略 | キーワード戦略、コンテンツ計画、内部施策 |
| 集客計画 | チャネルミックス、予算配分、KPI設計 |
| コンテンツ戦略 | ブログ、SNS、動画のコンテンツ計画 |

## 5つのレビュー視点

### 視点1: ターゲットユーザー（検索者・閲覧者）
- 検索意図と提供コンテンツが合致しているか
- ファーストビューで「自分向けだ」と感じるか
- 知りたい情報にすぐたどり着けるか
- 行動（電話・申込・問い合わせ）する動機が十分か
- モバイルでのユーザー体験は快適か

### 視点2: CROスペシャリスト（コンバージョン最適化）
- CVR（コンバージョン率）を上げる要素が揃っているか
- CTA（行動喚起）の配置・文言・デザインは適切か
- 離脱ポイントはどこか
- 信頼構築要素（実績・口コミ・資格・メディア掲載）は十分か
- フォームの入力ハードルは適切か

### 視点3: SEO・広告の専門家（集客効率）
- ターゲットキーワードの選定は適切か
- 検索ボリュームと競合度のバランス
- 広告のターゲティング設計は的確か
- CPA（獲得単価）は現実的か
- チャネル間の整合性（SEO × 広告 × SNS）

### 視点4: 競合分析の視点
- 競合と比較して差別化できているか
- 競合が持っていて自社にない要素はあるか
- 価格・品質・訴求ポイントでの優位性
- ユーザーが比較検討したときに選ばれるか

### 視点5: データアナリスト（計測・改善可能性）
- KPIは適切に設定されているか
- 計測環境は整っているか（GA4, GTM, CV計測）
- A/Bテストが実施しやすい設計か
- PDCAを回せる構造になっているか
- データに基づいた判断ができる設計か

## スコアリング基準

各視点を20点満点、合計100点満点で採点する。

```
## スコアリング

| 視点 | 現状スコア | 主な課題 | 改善後スコア |
|------|-----------|---------|-------------|
| ユーザー視点（検索意図） | __/20 | ... | __/20 |
| CRO視点（CV最適化） | __/20 | ... | __/20 |
| 集客視点（SEO・広告） | __/20 | ... | __/20 |
| 競合視点（差別化） | __/20 | ... | __/20 |
| データ視点（計測・改善） | __/20 | ... | __/20 |
| **合計** | **__/100** | | **__/100** |
```

### スコアの目安

| スコア | 評価 | 判定 |
|--------|------|------|
| 85-100 | 優秀。すぐに実行・公開してよいレベル | ✅ GO |
| 70-84 | 良好。微調整で実行可能 | ⚠️ 改善推奨 |
| 50-69 | 要改善。設計の見直しが必要 | 🔄 要修正 |
| 0-49 | 大幅な見直しが必要 | ❌ 再設計 |
