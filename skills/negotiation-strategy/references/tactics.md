# 実行フレームワーク・リスク管理・成果測定

## 1. 実行プラン詳細テンプレート

### フェーズ1: 事前準備

**目標設定**
| 項目 | 内容 | 数値/条件 |
|------|------|----------|
| 理想ライン | 最も望ましい結果 | |
| 妥協ライン | 受け入れ可能な最低条件 | |
| 撤退ライン | これ以下なら決裂を選ぶ | |
| BATNA | 交渉決裂時の最良の代替案 | |

**情報収集チェックリスト**
- [ ] 相手の組織構造・意思決定プロセス
- [ ] 過去の取引条件・交渉パターン
- [ ] 業界の標準的な条件・相場
- [ ] 競合他社の動向・提案状況
- [ ] 相手の財務状況・予算制約
- [ ] キーパーソンの個人的関心・価値観

**シナリオプランニング**
| シナリオ | 相手の出方 | 対応策 |
|---------|-----------|--------|
| 最良 | 即座に合意 | 追加条件の確認、文書化 |
| 標準 | 条件交渉 | 段階的譲歩計画の実行 |
| 困難 | 強硬な反対 | BATNA提示、一時中断 |
| 最悪 | 交渉決裂 | 代替案の実行、関係維持 |

**心理的準備**
- [ ] 感情トリガーの自己認識（怒り・焦り・不安の引き金）
- [ ] 深呼吸・冷静化のルーティン設定
- [ ] 「最悪のケース」の受容と心理的安全確保
- [ ] 自信を持てる根拠・実績の確認

### フェーズ2: 交渉実行

**オープニング（0-15分）**
- ラポール形成（共通点の発見、軽い会話）
- アジェンダの確認と合意
- 相手の現状・関心事の確認
- アンカリング（最初の提案の設定）

**中盤（メインの議論）**
- 情報開示のタイミング管理（段階的に）
- 認知バイアス活用の実行
- 譲歩の計画的実施
  - 各譲歩に条件を付ける（「○○するなら△△を」）
  - 譲歩の幅を徐々に小さくする
  - 重要でない項目から譲歩を始める
- 相手の反応の観察と戦略修正

**終盤（クロージング）**
- 合意内容の要約と確認
- 未解決項目のリスト化
- 次のステップの明確化
- ピーク・エンド効果を意識した締めくくり
  - ポジティブな将来像を共有
  - 感謝の表明
  - 具体的な次のアクションを設定

### フェーズ3: フォローアップ

- [ ] 合意内容の文書化（24時間以内）
- [ ] 関係者への報告・共有
- [ ] 履行状況の確認スケジュール設定
- [ ] 関係性維持のためのコミュニケーション計画
- [ ] 交渉プロセスの振り返り・学習記録

## 2. リスク管理フレームワーク

### BATNA（最良の代替案）管理

**BATNA評価シート**
| 評価項目 | 現在のBATNA | 改善後のBATNA |
|---------|------------|--------------|
| 代替案の具体性 | | |
| 実現可能性 | | |
| 魅力度（1-10） | | |
| 時間的制約 | | |
| コスト | | |

**BATNA強化策**
1. 代替の取引先・パートナーの開拓
2. 自社内での代替手段の検討
3. 第三者（仲介者・調停者）の活用
4. 時間的余裕の確保
5. 情報の追加収集

### 危険信号の早期検知

**即座に戦略変更が必要なサイン**

| 危険信号 | 対応策 |
|---------|--------|
| 相手が感情的（怒り・焦り） | 一時中断を提案、クールダウン期間設定 |
| 決裁権者が突然変更 | 新たな権者の利害を再調査 |
| 市場環境の急変 | 情報更新し、条件を再評価 |
| 競合他社の介入 | 差別化ポイントの再強調 |
| 法的・規制的制約の発覚 | 専門家に相談、合法的範囲の確認 |
| 相手が「公正」を武器にし始める | 「何が公正か一緒に考えましょう」と対話 |
| 一方的な最後通告 | BATNAの再確認、時間稼ぎ |

### よくある失敗パターン

| パターン | 原因 | 予防策 |
|---------|------|--------|
| 過度な譲歩 | 準備不足、圧力への屈服 | 譲歩計画を事前に策定 |
| 感情的対立 | 個人攻撃、プライドの衝突 | 問題と人を分離、休憩の活用 |
| 情報不足 | 事前調査の不足 | 情報収集チェックリストの徹底 |
| 時間切れ | 締切の設定ミス | バッファタイムの確保 |
| 権限不足 | 事前確認の欠如 | 交渉前に権限範囲を明確化 |
| 合意後の反故 | 文書化の不備 | 合意内容の即時文書化 |
| Win-Lose追求 | 短期視点、関係軽視 | 長期関係の価値を常に意識 |

## 3. 成果測定指標

### 定量的成功指標

| 指標 | 測定方法 | 目標水準 |
|------|---------|---------|
| 目標達成率 | 設定目標に対する達成度（%） | 80%以上 |
| 時間効率 | 予定時間に対する実際の交渉時間 | 予定の120%以内 |
| コスト効率 | 交渉にかかった総コスト vs 得られた価値 | ROI 3倍以上 |
| 合意速度 | 初回提案から合意までの期間 | 想定の150%以内 |

### 定性的評価項目

| 項目 | 評価基準 | 評価（5段階） |
|------|---------|-------------|
| Win-Win度 | 双方にとって価値ある結果か | |
| 関係性維持度 | 交渉後の相手との関係 | |
| 持続性 | 長期的に維持可能な合意か | |
| 学習度 | 今後の交渉に活かせる学びがあったか | |
| 総合満足度 | 双方の満足度 | |

## 5. 情報収集の優先順位

| 優先度 | 項目 | 理由 |
|--------|------|------|
| 最重要（必須） | 交渉目標、BATNA、相手の権限、時間制約 | これなしでは戦略立案不可 |
| 重要 | 過去の関係、業界慣行、市場環境、文化的要因 | 戦略の精度を大幅に向上 |
| 有用 | 個人的背景、組織風土、競合状況、規制環境 | 微調整に活用 |
