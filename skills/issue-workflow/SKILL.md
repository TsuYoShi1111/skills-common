# ISSUE案件提案書作成

## 概要

エンジニアマッチング案件の提案書を、統一フォーマットで作成する。

## 必須項目

以下の項目を必ず含める：

1. 案件名（プロジェクト名）
2. 案件概要（3行以内で簡潔に）
3. 求めるスキル
4. 勤務条件
5. 開始時期
6. 企業情報
7. 選考フロー

## 出力フォーマット

【案件名】
株式会社◯◯ / ××開発プロジェクト

【案件概要】
（3行以内で簡潔に記載）

【求めるスキル】
■必須スキル
-
-

■歓迎スキル
-
-

【勤務条件】
- 稼働：週◯時間（または週5日フルタイム）
- 期間：◯ヶ月〜（延長可能性あり）
- 単価：月額◯◯万円〜◯◯万円
- 勤務：リモート可/一部出社/フル出社

【開始時期】
◯月◯日〜 または 即日〜

【企業情報】
- 業種：
- 規模：

【選考フロー】
書類選考 → 面談 → 条件調整 → 参画

## 注意事項

- 単価は必ずレンジで記載（固定額は避ける）
- リモート可否は明確に記載
- 契約形態（業務委託/準委任/SES等）を明記
- 必須スキルと歓迎スキルは明確に分ける
