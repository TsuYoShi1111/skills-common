# 心理学・行動経済学 ビジネス活用ナレッジ

心理学・行動経済学の知識をビジネスシーンごとに整理し、実践的に活用する。

## 活用シーン別インデックス

### 営業・商談で使える心理学

| 原理 | 説明 | 営業での活用 |
|------|------|------------|
| 返報性の原理 | 何かをもらうとお返ししたくなる | 無料相談・サンプル提供→成約率UP |
| 社会的証明 | 多くの人がやっていることは正しいと感じる | 「導入実績1,000社」で安心感を演出 |
| 権威性 | 専門家や権威者の意見を信じやすい | 資格・メディア掲載実績の提示 |
| 希少性 | 限られたものほど価値を感じる | 「残り3枠」「今月限定」 |
| 一貫性の原理 | 一度YESと言うと次もYESと言いやすい | 小さなYESを積み重ねる（フット・イン・ザ・ドア） |
| アンカリング | 最初に提示された数字が基準になる | 高い見積→値引きで「お得感」 |

### マーケティング・広告で使える心理学

| 原理 | 説明 | 広告での活用 |
|------|------|------------|
| 損失回避 | 得する喜びより損する痛みが2倍強い | 「今やらないと損する◯◯」 |
| フレーミング効果 | 同じ内容でも表現で印象が変わる | 「成功率90%」vs「失敗率10%」 |
| バンドワゴン効果 | みんながやっていると自分もやりたい | 「累計10万人が利用」 |
| ピーク・エンドの法則 | 最も印象的な瞬間と最後の印象で全体を判断 | LPの最後にインパクトあるCTA |
| 選択のパラドックス | 選択肢が多すぎると決められない | プランは3つまでに絞る |
| デフォルト効果 | 初期設定のまま選びやすい | おすすめプランを目立たせる |

### マネジメントで使える心理学

| 原理 | 説明 | マネジメントでの活用 |
|------|------|-----------------|
| 自己効力感 | 「自分はできる」と思えると実際にできる | 小さな成功体験を積ませる |
| 内発的動機付け | 自分からやりたいと思う動機が最も強い | 自律性・達成感・成長を感じられる環境 |
| ピグマリオン効果 | 期待されるとパフォーマンスが上がる | 「あなたならできる」と信頼を伝える |
| 心理的安全性 | ミスを恐れず発言できる環境 | 失敗を責めず、チャレンジを称える |

### コンサルティングで使える心理学

| 原理 | 説明 | コンサルでの活用 |
|------|------|---------------|
| 認知バイアス | 思い込みによる判断の歪み | クライアントの思い込みを客観データで解消 |
| 確証バイアス | 自分に都合のいい情報だけ集める | 反証データも提示して視野を広げる |
| 現状維持バイアス | 変化を避け現状を続けたがる | 変化しないリスクを数値で示す |
| ナッジ | 選択を強制せず望ましい方向に誘導 | 選択肢の提示順序・デフォルト設定 |

## ナレッジの追加方法

新しく学んだ知識は以下の形式で追記する：

```
| 原理名 | 一行説明 | ビジネスでの活用法 |
```
